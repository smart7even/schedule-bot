Это бета версия, пока расписание есть только для ПИ 2002, но в скором будет добавлена функциональность для просмотра расписания для всех факультетов и групп СПбГЭУ

Команды, которые работают в диалоге с ботом
/schedule - Получить расписание любой группы. Нажмите на эту команду и название своей группы
/my_schedule - Получить расписание своей группы. Для работы этой команды нужно установить свою группу с помощью команды /set_group

Команды из любого чата
@schedule_unecon_bot <любые символы> - Получить расписание группы ПИ 2002
